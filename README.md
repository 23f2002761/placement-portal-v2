# Placement Portal Application - V2

A full-stack web application built using Flask, VueJS, SQLite, Redis, and Celery.

This system manages campus placements with role-based access for:
- Admin (Institute)
- Companies
- Students

Features include:
- Authentication and Role-Based Access
- Placement Drive Management
- Application Tracking
- Interview Scheduling
- Monthly Reports
- Async CSV Export
- Redis Caching

Built for Modern Application Development II.
